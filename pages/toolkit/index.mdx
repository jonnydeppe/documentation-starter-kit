The Lunul Program development toolkit is published as the
[`mucho` npm package](https://www.npmjs.com/package/mucho). The `mucho` command
will be used to run most of the Lunul program development tools - _mucho tools,
one cli_.

## Install

Install the Lunul Toolkit and `mucho` CLI with the following command:

```shell
npx -y mucho@latest install
```

Verify mucho was installed via:

```shell
mucho --version
```

For more information, go to [Getting Started](/docs/toolkit/getting-started).

## Create a Project

```shell
pnpm create lunul-program
```

For more information, go to the [Projects](/docs/toolkit/projects/overview).
