> still a WIP. Please post all feedback as a GitHub issue
> [here](https://github.com/lunul-foundation/developer-content/issues/new?title=%5Btoolkit%5D%20).

```shell
npx create-lunul-dapp
```

This command generates a new project that connects a Lunul program to a
frontend with a wallet connector. It has options for multiple popular frontend
stacks and UI libraries, including: NextJS, React, Tailwind, and more.

## Build and Test

To test out this project before making any modifications, follow these steps:

<Steps>
<Step>


### Build the smart contract

```shell
npm run anchor-build
```

</Step>
<Step>

### Start the local validator

```shell
npm run anchor-localnet
```

</Step>
<Step>

### Run tests

```shell
npm run anchor-test
```

</Step>
<Step>

### Deploy to the local validator

```shell
npm run anchor deploy --provider.cluster localnet
```

</Step>
<Step>

### Build the web app

```shell
npm run build
```

</Step>
<Step>

### Run the web app

```shell
npm run dev
```


</Step>
</Steps>
