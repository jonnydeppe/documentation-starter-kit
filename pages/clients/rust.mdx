## Rust Crates

The following are the most important and commonly used Rust crates for Lunul
development:

- [`lunul-program`] &mdash; Imported by programs running on Lunul, compiled to
  SBF. This crate contains many fundamental data types and is re-exported from
  [`lunul-sdk`], which cannot be imported from a Lunul program.

- [`lunul-sdk`] &mdash; The basic offchain SDK, it re-exports
  [`lunul-program`] and adds more APIs on top of that. Most Lunul programs
  that do not run on-chain will import this.

- [`lunul-client`] &mdash; For interacting with a Lunul node via the
  [JSON RPC API](/docs/rpc).

- [`lunul-cli-config`] &mdash; Loading and saving the Lunul CLI configuration
  file.

- [`lunul-clap-utils`] &mdash; Routines for setting up a CLI, using [`clap`],
  as used by the main Lunul CLI. Includes functions for loading all types of
  signers supported by the CLI.
