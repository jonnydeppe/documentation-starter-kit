h1: getVersion RPC Method
---

Returns the current Lunul version running on the node

<DocSideBySide>

<DocLeftSide>

### Parameters

**None**

### Result

The result field will be a JSON object with the following fields:

- `lunul-core` - software version of lunul-core as a `string`
- `feature-set` - unique identifier of the current software's feature set as a
  `u32`

</DocLeftSide>

<DocRightSide>

### Code sample

```shell
curl https://api.devnet.lunul.com -s -X POST -H "Content-Type: application/json" -d '
  {"jsonrpc":"2.0","id":1, "method":"getVersion"}
'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "result": { "feature-set": 2891131721, "lunul-core": "1.16.7" },
  "id": 1
}
```

</DocRightSide>
</DocSideBySide>
